## Info
Neste trabalho, o objetivo é `criar um conversor que parta de um ficheiro **MarkDown** e crie um ficheiro alternativo em **HTML**`, tendo apenas em conta os seguintes elementos:

1. Cabeçalhos
2. Bolds
3. Itálicos
4. Listas Numeradas (e não Numeradas) 
5. Links
6. Imagens
